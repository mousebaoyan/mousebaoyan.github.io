## 它解决什么问题

套磁季最耗时的往往不是写信本身，而是：翻遍官网找导师邮箱、用表格记录哪些发了哪些没发、不知道谁回了。

AutoEmailSender 把这些重复操作自动化——从导师抓取、匹配分析、草稿生成，到定时批量发送和回复追踪，整个流程在一个应用内完成。

> 系统替你完成信息整理和重复写信的工作，但最终发给谁、什么时候发、发什么内容，仍然由你来定。

## 主要功能

| 功能         | 说明                                           |
| ------------ | ---------------------------------------------- |
| 智能抓取     | 从学校官网整理导师信息，减少手动复制和表格维护 |
| 匹配度分析   | 结合你的材料和导师资料，辅助判断联系优先级     |
| 导师标签     | 自定义标签标记和筛选导师，支持导入导出         |
| 定时批量发送 | 草稿确认后立即发送或排定时间窗口，控制节奏     |
| 回复追踪     | 自动检测导师回复，标记联系状态，方便跟进       |
