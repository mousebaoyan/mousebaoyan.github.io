## 它解决什么问题

各学校报名系统长得不一样，但要填的信息基本相同——姓名、学校、GPA、手机号……每投一所就重复填一遍，费时又容易出错。

这个扩展让你**填一次资料，之后每到一个报名页面点一下自动填好**。

## 两种识别模式

**关键词规则模式**（默认，免费，离线）
靠内置关键词识别字段，所有资料只存本地、不联网。识别不到的字段会标红提示手动补。

**AI 模式**（可选，需 DeepSeek API Key）

调用 AI 读取整页字段智能匹配，能填规则识别不到的下拉框、特殊命名等。

开启后会把字段名和你的个人资料上传给 DeepSeek，介意隐私请只用关键词模式。

## 使用流程

1. 按 README 安装浏览器扩展（约1分钟，做一次即可）
2. 点扩展图标 → 设置我的资料 → 填好保存
3. 每到一个报名页面，点右下角「📝 一键填充」
4. 检查红框标注的未识别字段，手动补全后提交

## 注意

- 照片上传、验证码、登录密码需手动操作
- 提交前务必人工核对一遍，尤其是身份证、手机号、排名等关键信息
- AI 模式下核对更不能省
