# mousebaoyan.github.io

Astro static site scaffold for the `mousebaoyan.github.io` GitHub Pages site.

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

The production output is generated in `dist/`.

## Deploy

This repository includes a GitHub Pages workflow:

```text
.github/workflows/deploy.yml
```

Push to the `main` branch and configure GitHub Pages to use **GitHub Actions**.
